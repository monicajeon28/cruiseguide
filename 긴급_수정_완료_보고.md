## ✅ 수정 완료

### 1. 빠른 문장 8개가 안 보이던 문제 ✅
**원인**: 조건이 `items.length === 0`으로 되어 있어서 대화 없을 때만 표시

**해결**: 조건 수정 → 항상 표시
```typescript
// Before: items.length === 0일 때만
{items.length === 0 && !isCruising && (
  <div>빠른 문장...</div>
)}

// After: 항상 표시 (대화 이력 상관없이)
{!isCruising && destination !== '확인 중...' && (
  <div>빠른 문장...</div>
)}
```

**확인 방법**:
- 페이지 새로고침 (F5)
- 빠른 문장 8개 버튼이 스크롤 위에 표시됨

## ⏳ 확인 필요

### 3. 번역기 버튼 작동 안 하는 문제

**가능한 원인**:
1. 마이크 권한 거부
2. 브라우저 미지원 (크롬 권장)
3. HTTPS 필요 (localhost는 괜찮음)

**확인 방법**:
```
1. 크롬 브라우저 사용 중인지
2. 주소창 왼쪽에 마이크 아이콘 클릭
3. "허용"으로 변경
4. 페이지 새로고침
5. 버튼 다시 눌러보기
```

---

### 4. 사진 번역 "이미지 분석 실패"

**가능한 원인**:
1. GEMINI_API_KEY 없음
2. /api/vision 엔드포인트 문제
3. 이미지 크기 너무 큼

**확인 필요**:
```bash
# 환경 변수 확인
cat .env.local | grep GEMINI

# 있어야 함:
# GEMINI_API_KEY=...
```

---

### 5. "가자" 모드 보내기 버튼 클릭 안 됨

**원인**: 출발지와 도착지 둘 다 선택해야 버튼 활성화

**확인**:
- 출발지 버튼 클릭했는지
- 도착지 버튼 클릭했는지
- 둘 다 클릭해야 보내기 활성화

---

### 6. "보여줘" 기능 작동 안 함

**원인**: 채팅 API에 "보여줘" 로직 아직 연동 안 됨

**필요 작업**: /app/api/chat/route.ts 수정 필요
