> **작성일**: 2025-01-XX  
> **대상**: 50대 이상 크루즈 여행객  
> **목적**: 타이핑 불편, 시력 저하, 접근성 개선

## ✅ 이미 완료된 기능 (2025-01-XX)

### 1. 음성 입력 기능 (STT) ✅
- **기능**: 출발지/도착지 입력 필드에 마이크 버튼 추가
- **사용법**: 마이크 버튼 클릭 → 말하기 → 자동으로 텍스트 입력
- **효과**: 타이핑 어려움 해소, 빠른 입력
- **위치**: "어디서 출발하시나요?" 및 "어디에 도착하시나요?" 입력창 옆

### 2. 음성 출력 기능 (TTS) ✅
- **기능**: 지니 응답에 스피커 버튼 추가
- **사용법**: "듣기" 버튼 클릭 → 지니 응답을 음성으로 듣기
- **효과**: 눈의 피로 감소, 길을 가면서도 확인 가능
- **위치**: 지니 응답 메시지 하단

## 🎯 추가 기능 제안 (우선순위별)

### 1. 🔴 최우선: 첫 사용자 가이드 (Onboarding Guide)

**필요성**: ⭐⭐⭐⭐⭐ (매우 높음)  
**예상 개발 시간**: 3-4시간  
**난이도**: 🟡 보통

**제안 내용:**
- 처음 "지니야 가자" 탭 진입 시 자동으로 튜토리얼 표시
- 단계별 안내:
  1. "출발지" 입력하는 방법 (키보드 또는 마이크 버튼)
  2. "도착지" 입력하는 방법
  3. "보내기" 버튼 클릭
  4. 연관검색 버튼 활용법
  5. Google Maps 링크 사용법

**구현 방법:**
- 번역기의 `TranslatorTutorial` 컴포넌트 참고
- `localStorage`로 "이미 본 가이드" 체크
- 큰 버튼, 큰 텍스트, 애니메이션 화살표로 가리키기

**파일 위치:**
- `app/chat/components/GoAnywhere.tsx` (기존 컴포넌트 활용)
- 새 컴포넌트: `app/chat/components/GoModeTutorial.tsx`

---

### 2. 🟠 높은 우선순위: 검색 히스토리 (Search History)

**필요성**: ⭐⭐⭐⭐ (높음)  
**예상 개발 시간**: 2-3시간  
**난이도**: 🟡 보통

**제안 내용:**
- 최근 검색한 경로를 저장하여 다시 클릭 가능
- 예: "홍콩 공항 → 카이탁 터미널"을 3일 전에 검색했다면, "최근 검색" 섹션에 표시
- 클릭하면 자동으로 입력창에 채워짐

**구현 방법:**
- `localStorage`에 검색 기록 저장 (최대 10개)
- "최근 검색" 섹션을 InputBar 위에 표시
- 날짜/시간 표시 (예: "3일 전", "1시간 전")

**파일 위치:**
- `app/chat/components/InputBar.tsx`: 검색 기록 저장 로직
- 새 컴포넌트: `app/chat/components/SearchHistory.tsx`

---

### 3. 🟡 중간 우선순위: 자주 사용하는 경로 즐겨찾기 (Favorites)

**필요성**: ⭐⭐⭐ (중간)  
**예상 개발 시간**: 2-3시간  
**난이도**: 🟡 보통

**제안 내용:**
- "즐겨찾기" 버튼으로 자주 가는 경로 저장
- 예: "홈 → 인천공항", "홈 → 부산 크루즈 터미널"
- 한 번 클릭으로 빠르게 검색

**구현 방법:**
- 별 아이콘 버튼 추가 (검색 결과 또는 히스토리에서)
- `localStorage`에 즐겨찾기 저장
- "즐겨찾기" 섹션을 InputBar 위에 표시

**파일 위치:**
- `app/chat/components/Favorites.tsx` (새 컴포넌트)

---

### 4. 🟡 중간 우선순위: 대형 모드 (Large Mode)

**필요성**: ⭐⭐⭐ (중간)  
**예상 개발 시간**: 1-2시간  
**난이도**: 🟢 쉬움

**제안 내용:**
- 설정에서 "대형 모드" 켜기/끄기
- 버튼 크기 2배 증가
- 텍스트 크기 1.5배 증가
- 입력창 크기 증가

**구현 방법:**
- `localStorage`에 설정 저장
- Tailwind CSS 클래스 조건부 적용
- 예: `min-h-[70px]` → 대형 모드: `min-h-[140px]`

**파일 위치:**
- `app/chat/components/InputBar.tsx`: 조건부 스타일 적용
- `app/chat/components/Chips.tsx`: 조건부 스타일 적용
- 설정 페이지에 토글 버튼 추가

---

### 5. 🟢 낮은 우선순위: 고대비 모드 (High Contrast Mode)

**필요성**: ⭐⭐ (낮음)  
**예상 개발 시간**: 1-2시간  
**난이도**: 🟢 쉬움

**제안 내용:**
- 설정에서 "고대비 모드" 켜기/끄기
- 배경색과 텍스트 색상 대비 증가
- 눈의 피로 감소 (야외에서 사용 시 효과적)

**구현 방법:**
- Tailwind CSS 클래스 조건부 적용
- 예: `bg-white text-gray-800` → 고대비: `bg-black text-white`

**파일 위치:**
- 전역 설정으로 적용 (Tailwind `dark:` 모드 활용)

---

### 6. 🟢 낮은 우선순위: 오프라인 저장 (Offline Cache)

**필요성**: ⭐⭐ (낮음)  
**예상 개발 시간**: 3-4시간  
**난이도**: 🟡 보통

**제안 내용:**
- 자주 검색한 경로를 오프라인에서도 확인
- Google Maps 링크는 온라인 필요 (명시)

**구현 방법:**
- Service Worker로 캐싱
- 검색 결과를 `localStorage`에 저장
- "오프라인 모드" 안내 메시지 표시
