# 🐛 문제 해결 가이드

> **발견된 문제들과 해결 방법**

---

## 1. 빠른 문장이 안 보이는 문제

### 원인
- 조건이 `items.length === 0`으로 되어 있어서 대화가 없을 때만 표시
- 여행 정보가 없으면 표시 안 됨

### 해결
- 조건 수정: 항상 표시하도록 변경
- 파일 수정 중...

---

## 2. 번역기 버튼 작동 안 하는 문제

### 원인
- 음성 인식 권한 문제일 수 있음
- 브라우저 호환성 문제

### 해결 방법
```
1. 크롬(Chrome) 브라우저 사용
2. 마이크 권한 허용
3. HTTPS 환경 필요 (localhost는 괜찮음)
```

---

## 3. 사진 번역 "이미지 분석 실패" 오류

### 원인
- `/api/vision` 엔드포인트 문제
- Gemini API 키 문제
- 이미지 형식 문제

### 확인 필요
- `.env.local` 파일에 `GEMINI_API_KEY` 있는지
- 파일 크기가 너무 큰지

---

## 4. "가자" 모드에서 보내기 버튼 클릭 안 됨

### 원인
- InputBar 컴포넌트 문제
- 출발지/도착지 둘 다 선택해야 활성화

### 해결
- 코드 확인 필요

---

## 5. "보여줘" 기능 작동 안 함

### 원인
- 채팅 API에 "보여줘" 로직 추가 필요
- 아직 연동 안 됨

### 해결
- 채팅 API 수정 필요

---

## 6. 튜토리얼 확인 방법

### 방법
```
1. 브라우저 개발자 도구 열기 (F12)
2. Application > Local Storage
3. "hasSeenTranslatorTutorial" 삭제
4. 페이지 새로고침
→ 튜토리얼이 1초 후 표시됨
```
