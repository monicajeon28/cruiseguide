이제 로그인 후 보게 될 메인 대시보드 화면을, AI와의 대화에 집중하는 형태로 전면 재구성하자. `스크린샷 2025-09-18 011556.png` 이미지를 참고하되, 아래 지시를 정확히 따라줘.

1.  **메인 페이지 레이아웃 및 헤더 수정 (`src/app/page.tsx`)**:
    -   기존의 카드 그리드 레이아웃을 모두 제거하고, 페이지 전체를 채팅 중심으로 재구성해줘.
    -   페이지 상단 헤더에는 다음 텍스트만 표시해줘: **"[사용자 이름]님의 크루즈여행은 D-Day O일 남았습니다."** (DB에서 가져온 사용자 이름과 여행 정보로 동적으로 표시)
    -   헤더에서 **'[사용자 이름]님' 부분을 클릭**하면, 사용자의 정보를 볼 수 있는 **`/profile` 페이지로 이동**하도록 링크를 걸어줘. (지금은 빈 페이지만 만들어줘도 돼.)
    -   대시보드 하단에 있던 '새로운 여행 등록하기' 버튼은 **완전히 제거**해줘.

2.  **AI 채팅 UI를 화면 중심으로**:
    -   **`ChatUI` 컴포넌트**를 헤더 바로 아래, 화면의 **가장 넓은 중앙 영역을 차지**하도록 배치해줘.
    -   사용자가 로그인하면, 지니가 먼저 말을 걸도록 첫 번째 환영 메시지("어서오세요, [이름]님! 무엇을 도와드릴까요? ✨")가 자동으로 표시되게 해줘.

3.  **'+' 퀵메뉴 기능 구현**:
    -   화면의 한쪽 코너(예: 왼쪽 상단, 로고 옆)에 **'+' 모양의 아이콘 버튼**을 만들어줘.
    -   이 버튼을 클릭하면, 화면 옆에서 부드럽게 나타나는 **사이드 메뉴(또는 팝업 모달)**를 띄워줘.
    -   이 메뉴 안에 아래 기능들로 이동할 수 있는 버튼들을 배치해줘:
        -   `✅ 여행 준비 체크리스트`
        -   `💰 간단한 여행 가계부`
        -   `🏝️ 항구 투어 정보` (클릭 시 `/data/terminals.json`의 내용을 보여주는 페이지로 이동)
        -   `🌟 다른 분들의 여행 후기`

4.  **통번역 전용 플로팅 버튼(FAB) 구현**:
    -   화면의 **오른쪽 맨 아래**에, 다른 콘텐츠 위에 떠 있는 **빨간색(`brand-red`) 동그라미 버튼**을 고정(`position: fixed`)시켜줘.
    -   버튼 안에는 **스피커 모양 아이콘**을 넣어줘.
    -   이 버튼을 클릭하면, 이전에 우리가 설계했던 **'통합 통번역기' UI가 화면 전체를 덮는 모달(Modal)** 형태로 나타나도록 구현해줘.

5.  **백엔드 기능 구현**:
    -   확정된 모든 D-Day별 알림이 팝업 형태로 나타나도록 구현해줘.
    -   여행 종료 3일 후 계정이 자동으로 잠기는 Cron Job을 백엔드에 구현해줘.
